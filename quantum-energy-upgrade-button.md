## 🔬 技术实现

### 视觉层级（共6层）

```
Layer 0: 外层能量场光晕（::before）
         ↓
Layer 1: 按钮主体（深色金属质感）
         ↓
Layer 2: 能量流动扫描（::after）
         ↓
Layer 3: 文字和图标（发光效果）
         ↓
Layer 4: 金色角标主体
         ↓
Layer 5: 角标光芒（::before + ::after）
```

### 核心特效

#### 1. 能量场脉冲（energyPulse）
```css
- 周期：3秒
- 效果：径向光晕扩散 + 透明度变化
- 范围：按钮外围 300% 区域
- 模糊：12px 高斯模糊
```

#### 2. 能量流动（energyFlow）
```css
- 周期：2.5秒（悬停时加速至1.5秒）
- 效果：蓝色光带从左到右扫描
- 路径：-100% → +200%
- 强度：40% 透明度峰值
```

#### 3. 箭头浮动（arrowFloat）
```css
- 周期：2秒
- 效果：上下浮动 3px
- 滤镜：蓝色发光（6px drop-shadow）
```

#### 4. 角标浮动（badgeFloat）
```css
- 周期：3秒
- 效果：上下浮动 2px
- 阴影：多层立体阴影
```

#### 5. 光芒脉冲（glowPulse）
```css
- 周期：2秒
- 效果：金色光晕呼吸（60% → 100% 透明度）
- 缩放：1.0 → 1.2 倍
```

### 阴影系统

#### 按钮默认状态（3层）
```css
box-shadow:
  0 4px 12px rgba(59, 130, 246, 0.15),    /* 主阴影 - 蓝色光晕 */
  inset 0 1px 0 rgba(255, 255, 255, 0.05),  /* 内高光 - 顶部 */
  inset 0 -1px 0 rgba(0, 0, 0, 0.3);        /* 内阴影 - 底部 */
```

#### 按钮悬停状态（4层）
```css
box-shadow:
  0 8px 24px rgba(59, 130, 246, 0.3),     /* 主阴影 - 扩散 */
  0 0 40px rgba(147, 51, 234, 0.2),       /* 外发光 - 紫色 */
  inset 0 1px 0 rgba(255, 255, 255, 0.1), /* 内高光 - 增强 */
  inset 0 -1px 0 rgba(0, 0, 0, 0.4);      /* 内阴影 - 加深 */
```

#### 角标阴影（4层）
```css
box-shadow:
  0 4px 8px rgba(251, 191, 36, 0.4),      /* 金色光晕 */
  0 2px 4px rgba(0, 0, 0, 0.3),           /* 基础阴影 */
  inset 0 1px 0 rgba(255, 255, 255, 0.4), /* 内高光 */
  inset 0 -1px 0 rgba(0, 0, 0, 0.2);      /* 内阴影 */
```

## 🎬 动画细节

### 动画缓动函数

| 动画 | 缓动函数 | 说明 |
|-----|---------|------|
| 能量场脉冲 | ease-in-out | 自然呼吸感 |
| 能量流动 | linear | 匀速扫描 |
| 箭头浮动 | ease-in-out | 平滑往复 |
| 悬停变换 | cubic-bezier(0.4, 0, 0.2, 1) | Material Design 标准 |
| 角标浮动 | ease-in-out | 柔和浮动 |

### 性能优化

**GPU 加速属性：**
- ✅ `transform: translateY()` - 用于位置变化
- ✅ `opacity` - 用于透明度变化
- ✅ `filter: blur()` - 用于模糊效果

**避免使用：**
- ❌ `top/left/right/bottom` - 会触发重排
- ❌ `width/height` - 会触发重排
- ❌ `box-shadow` 动画 - 仅在必要时使用

## 🎯 交互状态

### 1. 默认状态（Idle）
```
视觉：深色金属质感 + 蓝色能量场脉冲
动画：
  - 外层光晕 3秒呼吸
  - 能量流 2.5秒扫描
  - 箭头 2秒浮动
  - 角标 3秒浮动 + 光芒脉冲
```

### 2. 悬停状态（Hover）
```
变化：
  - 边框：蓝色加强（0.3 → 0.6 透明度）
  - 阴影：扩散增强（12px → 24px）
  - 位置：上浮 2px
  - 能量流：加速（2.5s → 1.5s）
过渡：0.4s cubic-bezier
```

### 3. 激活状态（Active）
```
变化：
  - 位置：回落至 0px
  - 阴影：收缩（24px → 8px）
  - 内阴影：加深
过渡：瞬时反馈
```

## 💡 设计哲学

### 为什么选择能量流设计？

1. **产品属性驱动**
   - 储能柜 = 电能管理设备
   - 蓝色 = 电能的视觉化表达
   - 流动 = 能量传输的动态呈现

2. **用户心理**
   - 蓝色给人专业、可靠的感觉
   - 科技感增强产品价值感知
   - 动画暗示"系统正常运行"

3. **品牌一致性**
   - 与能源行业调性一致
   - 符合B2B专业形象
   - 易于扩展到其他组件

4. **长期可维护性**
   - 不会因流行趋势变化而过时
   - 蓝色科技感是永恒主题
   - 易于迭代优化
