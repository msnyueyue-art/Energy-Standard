## 修复内容

### 1. 恢复侧边栏滚动条 ✅

**需求:** 保持原有的侧边栏滚动条,不做隐藏处理

**操作:** 删除之前添加的滚动条隐藏样式

.sidebar {
    overflow-y: auto;
    scrollbar-width: none; /* Firefox */

### 2. 调整操作列按钮对齐 ✅

**需求:** 让"查看"、"编辑"、"删除"按钮向右移动,使"编辑"按钮与"操作"表头对齐

**修改位置:** [electricity-price-new.html:289-294](electricity-price-new.html#L289-L294)

**修改前:**
```css
.action-buttons {
    display: flex;
    gap: 8px;
}
```

**修改后:**
```css
.action-buttons {
    display: flex;
    gap: 8px;
    justify-content: flex-end;    /* 右对齐 */
    padding-right: 12px;          /* 右侧间距 */
}
```

**效果说明:**
- `justify-content: flex-end;` - 将按钮组整体右对齐
- `padding-right: 12px;` - 添加适当的右侧间距,使按钮不紧贴边缘
- 编辑按钮现在与"操作"表头标题对齐
