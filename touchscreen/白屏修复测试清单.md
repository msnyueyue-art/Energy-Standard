## 🛠️ 故障排查

### 如果仍有白屏

#### 检查1: 内联CSS是否生效
```javascript
// F12 控制台输入:
console.log(document.getElementById('critical-transition-css'));

// 应该输出:
// <style id="critical-transition-css">...</style>

// 如果是 null,说明CSS未加载
```

#### 检查2: body类是否添加
```javascript
// F12 控制台输入:
console.log(document.body.classList.contains('page-ready'));

// 应该输出: true
// 如果是 false,说明JS未执行
```

#### 检查3: CSS优先级
```javascript
// F12 控制台输入:
console.log(getComputedStyle(document.body).backgroundColor);

// 应该输出: "rgb(10, 14, 26)" 或类似深色
// 如果是白色,说明CSS被覆盖
```

#### 检查4: 文件加载
```
F12 → Network → 刷新页面
查找:
- page-transition.css (应该是 200 OK)
- page-transition.js (应该是 200 OK)
```

## ✅ 验收标准

### 必须满足 (P0)
- [ ] **零白屏**: 任何情况下都不出现白色背景
