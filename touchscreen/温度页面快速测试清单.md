## 🐛 常见问题排查

### 如果还显示中文:
1. ✅ 清除浏览器缓存 (`Ctrl + F5`)
2. ✅ 检查F12控制台是否有错误
3. ✅ 确认语言设置: F12 → Application → Local Storage → `touchscreen_language` = "en"
4. ✅ 尝试无痕模式测试

### 如果英文文本超出卡片:
1. ✅ 检查HTML的lang属性是否为"en"
2. ✅ 清除缓存后重新测试
3. ✅ 检查CSS是否正确加载
